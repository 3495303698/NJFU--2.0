# 基于局域网通信的简易聊天系统

一个基于Python和tkinter开发的简易聊天系统，支持局域网内用户通信、好友管理、即时消息和图片传输。

## 功能特性

- ✅ **图形界面**：使用tkinter构建友好的用户界面
- ✅ **通讯录功能**：按字母表顺序显示好友列表
- ✅ **添加好友**：通过服务器查询用户信息，发送好友请求，验证后添加
- ✅ **即时通信**：支持文字和图片消息传输，发送失败时显示提示
- ✅ **局域网发现**：自动发现局域网内其他在线用户，支持临时会话

## 系统架构

- `server.py` - 服务器端，处理用户注册、登录、好友请求、消息转发
- `client.py` - 客户端GUI，提供用户界面和通信功能
- `network.py` - 网络通信协议和UDP广播功能
- `contacts.py` - 通讯录管理模块
- `config.py` - 配置文件

## 使用方法

### 1. 启动服务器

```bash
python server.py
```

服务器默认监听 `0.0.0.0:8888`

### 2. 启动客户端

在多个终端/窗口中启动客户端：

```bash
python client.py
```

### 3. 使用流程

1. **注册/登录**
   - 首次使用需要注册账号（输入用户名和密码）
   - 已注册用户直接登录
   - 输入服务器地址（默认127.0.0.1）

2. **添加好友**
   - 点击"添加好友"按钮
   - 输入对方用户名和验证信息
   - 系统会查询用户信息并发送好友请求
   - 对方同意后，好友自动添加到通讯录

3. **聊天**
   - 双击通讯录中的好友，打开聊天窗口
   - 输入消息后点击"发送"或按Enter键
   - 点击"图片"按钮选择并发送图片

4. **局域网临时会话**
   - 在"局域网用户"列表中查看发现的在线用户
   - 双击用户可发起临时会话（无需添加好友）
   - 临时会话支持文字和图片传输

## 技术说明

- **TCP通信**：用于服务器与客户端的可靠数据传输
- **UDP广播**：用于局域网自动发现功能
- **JSON协议**：消息格式使用JSON编码
- **文件传输**：图片传输采用二进制流方式

## 数据存储

- 用户信息：`chat_data/users.json`
- 通讯录：`chat_data/contacts.json`
- 接收的图片：`chat_data/received_images/`

## 注意事项

1. 确保服务器先启动，客户端才能正常连接
2. 局域网发现功能需要客户端在同一局域网内
3. 图片发送功能在临时会话中完全支持，通过服务器的图片转发功能需要进一步实现
4. 客户端默认监听端口为7777，如端口被占用可能需要修改

## 系统要求

- Python 3.6+
- tkinter（通常随Python安装）
- 无额外依赖包
