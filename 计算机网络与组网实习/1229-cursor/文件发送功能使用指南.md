# 文件发送功能使用指南

## 功能概述
聊天系统已经完整实现了文件发送功能，支持多种文件类型的传输和下载。

## 支持的文件类型

### 🖼️ 图片文件
- .jpg, .jpeg, .png, .gif, .bmp, .webp

### 🎬 视频文件
- .mp4, .avi, .mov, .wmv, .flv, .mkv

### 🎵 音频文件
- .mp3, .wav, .flac, .aac, .ogg

### 📄 文档文件
- .pdf, .doc, .docx, .txt, .rtf, .odt

### 📦 压缩文件
- .zip, .rar, .7z, .tar, .gz

## 使用方法

### 1. 启动客户端
```bash
python client.py
```

### 2. 登录系统
- 输入用户名和密码登录

### 3. 开始聊天
- 选择联系人开始聊天会话

### 4. 发送文件
在聊天窗口的底部输入区域：
- 找到"文件"按钮（位于"发送"按钮旁边）
- 点击"文件"按钮
- 选择要发送的文件
- 文件会自动传输

### 5. 接收和下载文件
- 收到文件时，会显示文件信息
- 点击"[点击下载: 文件名]"文本
- 系统会自动打开文件所在目录

## 功能特点

### ✅ 已实现的功能
- **多文件类型支持**：图片、视频、音频、文档、压缩包
- **文件大小检查**：最大支持100MB
- **自动类型识别**：根据文件扩展名自动识别文件类型
- **实时大小显示**：显示文件大小的格式化信息（如1.2MB）
- **点击下载**：点击文本即可打开文件目录
- **平台兼容**：支持Windows、Mac、Linux系统
- **P2P传输**：支持临时会话的直接文件传输
- **服务器转发**：支持通过服务器的文件传输

### 📁 文件存储位置
- **发送的文件**：临时存储后自动传输
- **接收的文件**：保存到 `received_files` 目录

## 技术实现

### 核心组件
1. **UI界面**：client.py中的文件按钮和文件选择对话框
2. **网络协议**：network.py中的文件传输协议
3. **服务器支持**：server.py中的文件消息转发
4. **配置文件**：config.py中的文件类型和大小限制

### 网络协议
- **MSG_TYPE_FILE**：文件传输消息类型
- **文件元数据**：文件名、大小、类型、时间戳
- **文件内容**：二进制文件数据传输

## 测试方法

### 使用测试文件
我已经在 `test_files` 目录中创建了各种类型的测试文件：
- test_image.png
- test_document.txt
- test_audio.mp3
- test_video.mp4

### 测试步骤
1. 启动两个客户端实例
2. 在一个客户端中选择联系人
3. 点击聊天窗口中的"文件"按钮
4. 选择测试文件发送
5. 在另一个客户端中接收并下载文件

## 注意事项
- 文件大小不能超过100MB
- 仅支持指定格式的文件类型
- 接收的文件保存在received_files目录中
- 点击下载文本即可打开文件目录
