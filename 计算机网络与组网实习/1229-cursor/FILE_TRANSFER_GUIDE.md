# 文件传输功能使用指南

## 🎯 功能概述

聊天应用现在支持完整的文件传输功能，用户可以发送和接收各种类型的文件。

## 📋 支持的文件类型

### 图片文件
- **格式**：JPG, PNG, GIF, BMP, WEBP
- **特点**：支持预览显示

### 视频文件
- **格式**：MP4, AVI, MOV, MKV, WMV
- **特点**：大文件传输支持

### 文档文件
- **格式**：TXT, PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX
- **特点**：办公文档完全支持

### 音频文件
- **格式**：MP3, WAV, FLAC, AAC
- **特点**：音频文件传输

### 压缩文件
- **格式**：ZIP, RAR, 7Z, TAR
- **特点**：批量文件传输

## 🚀 使用方法

### 1. 启动服务器
```bash
python server.py
```
服务器将监听 0.0.0.0:8888

### 2. 启动客户端
```bash
python client.py
```

### 3. 发送文件
1. 登录或注册用户账户
2. 在聊天窗口中点击"发送文件"按钮
3. 选择要发送的文件（支持10MB以内）
4. 选择目标用户
5. 文件将自动传输

### 4. 接收文件
1. 当收到文件时，文件信息会显示在聊天窗口
2. 点击"下载"链接打开文件位置
3. 文件自动保存到 `chat_data/received_files/` 目录

## 🔧 技术特性

### 文件大小限制
- **最大文件大小**：10MB
- **超出限制**：自动提示并拒绝传输

### 文件类型识别
- 基于文件扩展名自动识别
- 支持文件类型图标显示
- 不同类型使用不同颜色区分

### 安全传输
- 通过加密socket连接传输
- 文件内容完整性验证
- 服务器端转发支持

### 存储管理
- 自动创建存储目录
- 防止文件名冲突
- 支持中文文件名

## 📁 文件存储结构

```
chat_data/

│   ├── document_1.txt

│   └── video_1.mp4
├── received_images/         # 图片文件存储

```

## 🛠️ 故障排除

### 文件无法发送
- 检查文件大小是否超过10MB
- 确认目标用户在线
- 检查网络连接

### 文件无法下载
- 确认文件存储目录存在
- 检查文件权限
- 尝试重新启动客户端

### 服务器错误
- 检查服务器是否正常运行
- 确认端口8888未被占用
- 查看服务器日志输出
