## 🌌 关于本仓库

这里不是简单的资料堆砌，而是一份**完整的计算机科学本科知识图谱**。从高等数学的公理化体系，到编译原理的底层抽象；从算法分析的计算思维，到机器视觉的智能感知——本仓库以课程为经、以实践为纬，编织出一张覆盖计算机科学与技术核心领域的学习网络。

在信息爆炸的时代，**结构化知识比碎片化信息更有力量**。本仓库致力于将本科阶段的每一门课程、每一份作业、每一次实验，沉淀为可追溯、可复用、可迭代的数字资产。
