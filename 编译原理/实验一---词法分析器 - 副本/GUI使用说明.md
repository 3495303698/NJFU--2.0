# 词法分析器 GUI 版本使用说明

## 程序概述

这是一个基于 Windows GUI 的词法分析器应用程序，用于编译原理实验一。程序提供了友好的图形界面，无需通过命令行操作。

## 主要功能

1. **文件选择**：通过浏览按钮选择源文件，或直接在输入框中输入文件路径
2. **词法分析**：对选定的源文件进行词法分析
3. **结果显示**：以表格形式显示分析结果，包括单词值、类型和行号
4. **结果保存**：将分析结果保存到文本文件

## 界面说明

### 1. 源文件路径区域
- **输入框**：可以手动输入文件路径，或通过浏览按钮选择
- **浏览按钮**：打开文件选择对话框，方便选择源文件
- **开始词法分析按钮**：执行词法分析操作

### 2. 分析状态信息区域
- 显示分析过程中的状态信息
- 显示分析结果统计（token数量、错误数量等）

### 3. 词法分析结果区域
- 以表格形式显示所有识别出的token
- 格式：`单词值 | 类型 | 行号`
- 支持滚动查看所有结果
- **保存结果按钮**：将当前分析结果保存到文件

## Token类型说明

程序识别以下类型的token：

- **保留字(1)**：main, printf, if, int, for, while, do, return, break, continue
- **标识符(2)**：变量名、函数名等
- **常数(3)**：整数和字符串常量
- **运算符(4)**：+, -, *, /, =, >, <, >=, <=, != 等
- **分隔符(5)**：, ; { } ( ) [ ] 等

## 使用方法

### 步骤1：编译程序

1. 打开 **Visual Studio 命令提示符**（Developer Command Prompt）
2. 使用 `cd` 命令导航到程序所在目录
3. 运行以下命令之一：
   - `编译GUI应用.bat`
   - `编译命令.bat`
   - `compile.bat`

### 步骤2：运行程序

编译成功后，双击生成的 `词法分析器.exe` 文件即可运行。

### 步骤3：进行词法分析

1. **选择源文件**：
   - 点击"浏览..."按钮，在文件对话框中选择要分析的源文件（如 program1.txt）
   - 或直接在输入框中输入文件路径

2. **开始分析**：
   - 点击"开始词法分析"按钮
   - 程序会自动进行词法分析

3. **查看结果**：
   - 在"分析状态信息"区域查看分析状态
   - 在"词法分析结果"区域查看详细的token列表

4. **保存结果**（可选）：
   - 点击"保存结果"按钮
   - 在保存对话框中选择保存位置和文件名
   - 结果将以文本格式保存

## 示例

### 示例源文件 (program1.txt)

```
main()
{
    int i,j,m,n;
    printf("please input data to i,j");
    i=1;
    j=4;
    m=i+j;
    n=i-j;
    printf("m=%d,n=%d",m,n);
}
```

### 分析结果示例

```
单词值                    类型            行号
--------------------------------------------------
main                      保留字(1)       1
(                         分隔符(5)       1
)                         分隔符(5)       1
{                         分隔符(5)       2
int                       保留字(1)       3
i                         标识符(2)       3
...
```

## 注意事项

1. **文件路径**：如果源文件不在程序目录下，需要提供完整路径
2. **文件格式**：源文件应为文本格式（.txt）
3. **字符编码**：建议使用 UTF-8 编码的文本文件
4. **错误处理**：如果文件无法打开或分析出错，会在状态信息区域显示错误提示

## 常见问题

### Q: 程序无法打开文件？
A: 请检查：
- 文件路径是否正确
- 文件是否存在
- 文件是否被其他程序占用

### Q: 分析结果为空？
A: 可能原因：
- 源文件中没有可识别的token
- 文件格式不正确
- 文件内容为空

### Q: 如何保存分析结果？
A: 点击"保存结果"按钮，在保存对话框中选择保存位置即可。

## 技术说明

- **GUI框架**：Win32 API
- **编译要求**：Visual Studio 或支持 Windows SDK 的编译器
