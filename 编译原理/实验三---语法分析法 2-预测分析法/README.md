# LL(1) 预测分析器

一个基于 Tkinter 的 LL(1) 预测分析器图形界面程序，用于演示和执行 LL(1) 语法分析过程。

## 功能特点

- **图形界面操作**：直观的用户界面，包括输入区域、分析结果显示和详细的分析过程表格
- **语法分析**：支持对表达式进行 LL(1) 预测分析，判断表达式是否合法
- **错误处理**：完善的输入验证和错误提示机制
- **分析过程可视化**：详细展示每一步的分析栈、剩余输入和使用的产生式
- **用户体验优化**：支持键盘快捷键、重置功能和实时反馈

## 支持的文法

程序使用以下 LL(1) 文法规则：

```
E → TE'
E' → +TE' | ε
T → FT'
T' → *FT' | ε
F → (E) | i
```

其中：
- `E`: 表达式
- `E'`: 表达式尾部（包含加法操作）
- `T`: 项
- `T'`: 项尾部（包含乘法操作）
- `F`: 因子
- `i`: 标识符
- `+`, `*`, `(`, `)`: 运算符和括号

## 使用方法

1. 确保安装了 Python 3.x
2. 运行主程序：
   ```
   python main.py
   ```
   或者直接运行：
   ```
   python gui.py
   ```
3. 在输入框中输入要分析的符号串，确保以 `#` 结尾（如果未添加，程序会自动补充）
4. 点击"LL(1)分析"按钮开始分析
5. 查看分析结果和详细分析过程

## 示例输入

- 简单表达式：`i#`, `i+i#`, `i*i#`
- 复杂表达式：`(i+i)*i#`, `i*(i+i)#`, `(i*i+i)#`
- 非法表达式：`i++i#`（连续运算符）, `(i)#`（多余括号）, `i(i)#`（缺少运算符）

## 键盘快捷键

- **Enter**: 执行分析
- **Ctrl+R**: 重置界面
- **Ctrl+Q**: 退出程序
- **Tab**: 在输入框和按钮间循环切换焦点

## 文件结构

- **main.py**: 程序入口文件
- **gui.py**: GUI 界面实现
- **ll1_parser.py**: LL(1) 分析器核心算法
- **test_ll1_parser.py**: 测试脚本
- **README.md**: 程序说明文档

## 错误处理

程序可以检测并报告以下错误：

- 空输入
- 不匹配的括号
- 非法字符
- 不符合文法规则的表达式结构
- 缺少结束符号 `#`

## 开发说明

本程序基于 Python 的 Tkinter 库构建，实现了完整的 LL(1) 预测分析算法和友好的用户界面。程序采用了面向对象的设计方法，将分析器核心算法与界面展示分离，便于维护和扩展。
