# 测试说明

## 测试方法

1. 编译程序后，运行 `parser.exe`
2. 按照提示输入测试用例
3. 验证输出结果是否符合预期

## 测试用例说明

### 合法符号串（应输出"合法符号串"）

- `i+i#` - 简单加法
- `i-i#` - 简单减法
- `i*i#` - 简单乘法
- `i/i#` - 简单除法
- `i+i*i#` - 混合运算
- `i+i/i#` - 混合运算
- `(i+i)*i#` - 带括号的表达式
- `i*(i+i)#` - 带括号的表达式
- `(i)#` - 括号包裹单个标识符
- `i#` - 单个标识符

### 非法符号串（应输出"非法符号串"及错误信息）

- `i+i*#` - 运算符后缺少操作数
- `i++#` - 连续运算符
- `(i+i#` - 括号不匹配（缺少右括号）
- `i+i)#` - 括号不匹配（多余的右括号）
- `(i))#` - 括号不匹配（多余的右括号）
- `i*i*i*#` - 末尾缺少操作数
- `+#` - 表达式为空
- `*#` - 表达式为空
- `#` - 空表达式

## 自动化测试

可以使用以下方式批量测试：

```bash
# Windows PowerShell
Get-Content test\test_inputs.txt | ForEach-Object {
    if ($_ -notmatch '^#' -and $_.Trim() -ne '') {
        echo $_ | .\parser.exe
    }
}
```
